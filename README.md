# hs-test-task | ![CI](https://github.com/ApricotLace/hs-test-task/workflows/CI/badge.svg) | [Demo on heroku](https://boiling-castle-62493.herokuapp.com/) | [Demo on KubeSail](https://patients.apricotlace.usw1.kubesail.io/)
## **Stack**
* **Ring-jetty-adapter** - server
* **Compojure** - routing
* **Hiccup** - html templating
* **Formative** - form validators
* **Jdbc** - db api 
* **Honeysql** - sql query builder

## Build & run
`sudo docker-compose up` &
`open localhost:7000`
